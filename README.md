### 介绍
本模块会以SuperSU或者Magisk的形式，来将原基于Nexus的Adreno GPU 3xx/4xx 的驱动镜像刷入设备。

### 注意
本模块的版本号会根据已刷入的Adreno GPU驱动版本号的变动而变动。
